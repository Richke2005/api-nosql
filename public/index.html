Schemaless API Documentation

Node.js | Express | MongoDB

Overview

Welcome to the Schemaless API documentation. This API leverages MongoDB's schema-less design to provide flexibility for storing and retrieving data.

Base URL: http://localhost:3000/api-nosql/api/v1/

Features

  • Manage users and products with full CRUD operations.
  • Search functionality for flexible data queries.
  • Relational data retrieval between users and products.

Getting Started

Prerequisites

  • Node.js (14+ recommended)
  • npm (included with Node.js)
  • MongoDB (local or remote like MongoDB Atlas)

Setup

  1. Clone the repository: 
    git clone <repository-url>
    cd <repository-folder>
  2. Install dependencies: 
    npm install
  3. Set up environment variables: 
    cp .env.example .env
    Update .env with your MongoDB connection string and other settings.
  4. Run the server: 
    npm run dev
    or 
    node server.js

Endpoints

Users

  • Create User: POST /users
    • {
    "name": "John Doe",
    "email": "john.doe@example.com",
    "password": "password123"
}
  • Get All Users: GET /users
  • Get User by ID: GET /users/:id
  • Update User: PUT /users/:id
  • Delete User: DELETE /users/:id

Products

  • Create Product: POST /products
  • Get All Products: GET /products
  • Get Product by ID: GET /products/:id
  • Search Products: GET /products/search
  • Update Product: PUT /products/:id
  • Delete Product: DELETE /products/:id

Example Usage

GET Request

curl -X GET http://localhost:3000/api-nosql/api/v1/users
